🔄 端末間同期の設定

Firebase設定
同期コード
接続状態
Firebaseの無料プランでリアルタイム同期できます。
以下の手順で設定してください（5分程度）。
  1. Firebase Console を開き、「プロジェクトを追加」
  2. プロジェクト名を入力 → 作成完了まで待つ
  3. 左メニュー「構築」→「Realtime Database」→「データベースを作成」→「テストモードで開始」を選択
  4. プロジェクトの概要 → 「</>」(ウェブ) → アプリを登録 → firebaseConfig の JSON をコピー
  5. 以下のテキストエリアに貼り付けて「保存」
同じ同期コードを使うと、複数の端末でスケジュールが共有されます。
自分だけのキーワード（例: tanaka-schedule）を設定してください。
⚠️ コードを知っている人は全員データを閲覧・編集できます。推測されにくい文字列にしてください。
